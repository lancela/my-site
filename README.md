# my-site

这是一个现代化的电商独立站项目，采用 Vite + Tailwind CSS 构建。

## 功能特性

- 🚀 Vite 快速构建
- 🎨 Tailwind CSS 样式
- 📝 Markdown 博客支持
- 🔍 SEO 优化（JSON-LD、Sitemap）
- 📱 响应式设计
- ⚡ 性能优化

## 快速开始

```bash
npm install
npm run dev
npm run build
```

## 项目结构

- `src/pages/` - HTML 页面
- `src/layouts/` - 布局组件
- `src/styles/` - 样式文件
- `src/js/` - JavaScript 脚本
- `src/content/posts/` - 博客内容（Markdown）
- `scripts/` - 构建脚本
- `public/` - 静态资源
