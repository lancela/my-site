欢迎来到我们的博客

这是第一篇博客文章。我们使用 Markdown 格式编写内容，然后通过构建脚本自动转换为 HTML。

主要特性

  • 📝 Markdown 支持
  • 🎨 自动样式应用
  • 📱 完全响应式
  • ⚡ 快速加载

如何使用

  1. 在 src/content/posts/ 目录创建新的 .md 文件
  2. 运行 npm run build 构建项目
  3. 生成的 HTML 文件会自动放在 src/pages/blog/ 目录

技术栈

我们使用：

  • Vite 作为构建工具
  • Tailwind CSS 进行样式
  • Marked 解析 Markdown

希望你喜欢这个博客系统！
